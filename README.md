# 🚀 My Portfolio

Welcome to my personal portfolio website! This project serves as a central hub to introduce who I am, showcase my professional experience and featured projects, highlight my academic background, and detail my technical skill set.

---

## 🛠️ Tech Stack & Tools

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **UI Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Package Manager & Runtime:** [Bun](https://bun.sh/)

---

## 💻 Getting Started

This project uses **Bun** for fast package management and execution. Follow the commands below to run the project locally.

### Development Mode
To start the local development server with hot-reloading:
```bash
bun dev
